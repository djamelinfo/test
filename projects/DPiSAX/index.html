DPiSAX: Massively Distributed Partitioned iSAX

Distributed Partitioned iSAX for Time Series Indexing in Massively Distributed Environments.

Requirements

Distributed Partitioned iSAX works with Apache Spark. In order to run DPiSAX you must download and install Apache Spark 1.6.2.

Building

The code is written in Java and We use maven to build it, Use the given pom.xml file to build an executable jar containing all the dependencies.

Datasets

We carried out our experiments on synthetic datasets using a random walk data series generator, each data series consisting of 256 points. At each time point the generator draws a random number from a Gaussian distribution N(0,1), then adds the value of the last number to the new number. You can use Random Walk Time Series Generator to produce a set of randomWalk datasets.
